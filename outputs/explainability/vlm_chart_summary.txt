# What Each Chart Shows
1. **Confusion Matrix**: This chart displays the classification performance of the model across three classes: `<30`, `>30`, and `NO`. It shows the true labels versus the predicted labels to illustrate where the model is making correct and incorrect predictions.
   
2. **Per-Class F1 Scores**: This chart represents the F1 scores for each class. The F1 score is a measure of a model's accuracy that considers both precision and recall, providing a balance between the two metrics.

# Class-Imbalance Impact and Error Pattern
- There is a notable class imbalance where the `NO` class dominates the predictions, leading to a higher F1 score. The `<30` class suffers heavily from low prediction quality, with only a few correct classifications.
- The model struggles most with the `<30` class, indicating a pattern of underestimating this category, which can affect clinical outcomes in a real-world setting.

3. **Model Selection**: Experiment with different algorithms (e.g., ensemble methods) that may perform better on imbalanced datasets.

5. **Cross-Validation**: Apply stratified cross-validation to ensure that all classes are well represented in training and validation sets.
